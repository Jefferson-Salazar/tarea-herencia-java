# Tarea Herencia Java

## Descripcion
En esta tarea se trabajó el tema de herencia en Java, usando clases padre e hijas.
Tambien se aplicó la sobrescritura de métodos con `@Override`, el uso de `super`
y atributos con acceso `protected`.

## Ejercicios realizados
1. Persona, Docente y Estudiante
2. Cuenta y CuentaCorriente
3. Animal y Perro
4. Empleado y Gerente
5. Vehiculo y Moto
6. Transporte y Bus
7. Producto y ProductoPerecedero
8. Figura, Rectangulo y Circulo
9. Libro y LibroDigital
10. Sistema de facturación

## Conceptos aplicados
- Herencia
- Sobreescritura de métodos
- Uso de `super`
- Encapsulamiento
- Validaciones en métodos

## Pruebas
Las pruebas de cada ejercicio se realizaron en la clase `Main`, creando objetos
y llamando sus métodos para verificar el funcionamiento.
